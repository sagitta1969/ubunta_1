задание 1 
Просмотреть содержимое директорий /etc, /proc, /home. 
Посмотреть пару произвольных файлов в /etc.
команды
~$ cd / "вошли в корневую директорию"
~$ ls -la "просмотр содержимого, 483 916 файлов и каталогов"
~$ cd /etc "в директори etc"
~$ ls -la "просмотр директории etc - 1 092 фалов и каталогов"
~$ cd /proc "в директори proc"
~$ ls -la "просмотр директории proc - 4 фалов и каталогов"
~$ cd /home "в директори home"
~$ ls -la "просмотр директории home - 12 фалов и каталогов"

для посмотра произвольных файлов директории /etc
используем команду cat
/etc$ cat timezone
Europe/Moscow
/etc$ cat hostname
ubuntu-VirtualBox
/etc$ cat protocols
# internet (IP) protocols

задание 2
cat это сокращение от слова catenate, задача команды - она читает данные из файла 
или стандартного ввода и выводит их на экран.
синтаксис команды: $ cat опции файл1 файл2 ...
создание файлов с помощью команды cat
~/max$ cat >test1
nvjhjhfjvkkfvbkjhfh "набираем произвольный текст"
нажмаем Ctrl + d
~/max$ cat test2 "смотрим содержимое файла"
oiwugutsnnbu
повторяем команды со вторым файлом test2
~/max$ cat test1 test2 > test3 "объединяем содержимое файлов в новом файле test3"

~/max$ mv -v test3 TEST "переименовываем файл test3 на TEST, 
renamed 'test3' -> 'TEST' опция  -v показывает список обработанных объектов"
